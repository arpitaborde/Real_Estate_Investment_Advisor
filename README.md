# 🏠 Real Estate Investment Advisor

## Project Overview

The Real Estate Investment Advisor is a Data Analytics project designed to help investors, buyers, and real estate businesses make informed property decisions through data-driven insights.

Using a dataset of 250,000+ property listings across multiple Indian states and cities, the project performs comprehensive data cleaning, exploratory data analysis (EDA), feature engineering, and interactive dashboard development.

The dashboard enables users to explore property trends, compare locations, analyze pricing patterns, evaluate amenities, and identify potential investment opportunities.

---

## Objectives

* Analyze property prices across cities and states.
* Identify factors affecting property valuation.
* Explore the impact of amenities, security, transport access, and property characteristics.
* Build an interactive dashboard for real-time exploration of real estate data.
* Generate business insights for investment decision-making.
